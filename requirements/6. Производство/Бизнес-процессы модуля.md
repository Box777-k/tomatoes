| Номер | Название бизнес-процесса | Краткое описание |
|-------|--------------------------|------------------|
| БП-6.1 | Открытие производственного цикла | Пользователь создаёт новый производственный цикл (партию выращивания) с указанием культуры (томаты, огурцы), теплицы, плановой даты начала и окончания. Цикл получает уникальный номер и статус "Открыт". Все последующие затраты (материалы, труд, косвенные расходы) привязываются к этому циклу для накопления на счёте 20 "Незавершённое производство". Цикл служит объектом калькуляции себестоимости продукции. |
| БП-6.2 | Списание материалов в производство | Пользователь оформляет документ списания материалов со склада в производство (семена, удобрения, средства защиты растений, тара, топливо). Документ привязывается к производственному циклу и указывается центр затрат. При проведении система уменьшает складские остатки и увеличивает стоимость незавершённого производства. Материальные затраты учитываются как прямые затраты цикла и полностью включаются в себестоимость продукции. |
| БП-6.3 | Отражение трудовых затрат в производстве | Пользователь регистрирует затраты на оплату труда работников, занятых в производственном цикле (агрономы, рабочие теплиц). Трудозатраты привязываются к производственному циклу и периоду начисления. Система относит эти расходы на прямые затраты производства. Данные используются для расчёта себестоимости и анализа структуры затрат по статье "Оплата труда". |
| БП-6.4 | Распределение косвенных производственных расходов | В конце периода система или пользователь распределяет общепроизводственные расходы (электроэнергия, отопление теплиц, амортизация оборудования) между открытыми производственными циклами. Распределение выполняется по заданной базе (площадь теплицы, количество растений, прямые затраты). Результаты распределения увеличивают стоимость незавершённого производства каждого цикла. Правила и история распределения сохраняются для аудита. |
| БП-6.5 | Фиксация потерь и брака в производстве | Пользователь регистрирует потери продукции на этапе выращивания или сбора (порча, болезни растений, механические повреждения при сборке). Указывается производственный цикл, количество потерь и причина. Система относит стоимость потерь на себестоимость продукции или на отдельный счёт непроизводственных расходов в зависимости от причины. Потери учитываются при расчёте выхода продукции и анализе эффективности. |
| БП-6.6 | Выпуск готовой продукции из производства | Пользователь оформляет документ выпуска готовой продукции при завершении производственного цикла или его этапа. Указывается количество выпущенной продукции, качество (сорт, класс), производственный цикл. При проведении система переводит накопленные затраты из незавершённого производства (счёт 20) в готовую продукцию (счёт 43). Рассчитывается себестоимость единицы продукции. Продукция оприходуется на склад готовой продукции с партионным учётом. |
| БП-6.7 | Партионный учёт готовой продукции | Система автоматически создаёт партию продукции при выпуске с привязкой к производственному циклу и дате. Партия содержит информацию о количестве, себестоимости, качестве и происхождении продукции. Партии используются для прослеживаемости при продажах и возвратах. При реализации система списывает товар с конкретных партий по методу ФИФО или по средней себестоимости согласно учётной политике. |
| БП-6.8 | Закрытие производственного цикла | После полного выпуска продукции и завершения всех операций пользователь закрывает производственный цикл. Система выполняет окончательный расчёт себестоимости, проверяет корректность отнесения затрат, формирует калькуляцию. Закрытый цикл становится недоступен для изменений без специальных прав. Данные закрытого цикла используются для анализа эффективности и планирования будущих циклов. |
| БП-6.9 | Расчёт и анализ себестоимости продукции | Система автоматически рассчитывает себестоимость продукции по каждому производственному циклу как сумму прямых материальных затрат, трудозатрат и распределённых косвенных расходов. Пользователь может просмотреть детальную калькуляцию с разбивкой по статьям затрат. Себестоимость рассчитывается как для всего цикла, так и в пересчёте на единицу продукции. Данные используются для ценообразования и контроля рентабельности производства. |
| БП-6.10 | Формирование производственной отчётности | Система формирует отчёты по производству: объём выпуска по периодам и культурам, структура затрат, динамика себестоимости, процент потерь, эффективность использования материалов. Отчёты строятся в разрезе производственных циклов, теплиц, периодов. Данные сопоставляются с плановыми показателями. Отчёты используются руководством для принятия управленческих решений и оптимизации производственных процессов. |
