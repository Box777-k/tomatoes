| Номер | Краткое название функции приложения (web app) | Описание в 3–4 предложения                                                                                                                                                                                                                                                                  |
| ----- | --------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 7.1   | Карточка основного средства                   | Пользователь создаёт и ведёт карточку основного средства с привязкой к организации. Карточка используется для учёта, амортизации и аналитики. Основное средство может быть активным, законсервированным или выведенным из эксплуатации. История изменений по объекту должна сохраняться.    |
| 7.2   | Группы и классификация ОС                     | Пользователь настраивает группы основных средств для классификации и отчётности. Группы используются для назначения сроков полезного использования и методов амортизации. Изменение группы не должно влиять на уже начисленную амортизацию. Группы нельзя удалять при наличии связанных ОС. |
| 7.3   | Объекты капитальных вложений                  | Пользователь создаёт объект капвложений для накопления затрат на счёте 07. Объект отражает строительство, модернизацию или приобретение ОС. Все затраты привязываются к конкретному объекту капвложений. Объект закрывается при вводе ОС в эксплуатацию.                                    |
| 7.4   | Накопление затрат по капвложениям             | Система аккумулирует все затраты, связанные с объектом капвложений. Затраты могут поступать из документов, услуг, зарплаты и аренды. Пользователь видит текущую стоимость и состав затрат. Должна поддерживаться детализация по статьям и контрагентам.                                     |
| 7.5   | Ввод основного средства в эксплуатацию        | Пользователь оформляет ввод ОС на основании объекта капвложений или прямой покупки. При вводе формируется первоначальная стоимость и бухгалтерская операция. После ввода капвложения закрываются. Операция должна быть необратимой без специальных прав.                                    |
| 7.6   | Начисление амортизации                        | Система начисляет амортизацию по основным средствам по заданным правилам. Начисление выполняется периодически и формирует бухгалтерские проводки. Пользователь должен видеть сумму амортизации и остаточную стоимость. Начисление должно учитывать статус ОС.                               |
| 7.7   | Распределение амортизации                     | Амортизация может распределяться по центрам затрат, подразделениям или видам деятельности. Пользователь задаёт правила распределения. Результат распределения влияет на себестоимость и управленческую аналитику. История распределений должна сохраняться.                                 |
| 7.8   | Модернизация и улучшение ОС                   | Пользователь отражает затраты на модернизацию и улучшение основных средств. Эти затраты увеличивают стоимость ОС и могут влиять на срок полезного использования. Операции оформляются через объект капвложений. Должна сохраняться история модернизаций.                                    |
| 7.9   | Выбытие основного средства                    | Пользователь оформляет выбытие ОС (продажа, списание, передача). Операция формирует соответствующие бухгалтерские проводки. Пользователь должен видеть финансовый результат выбытия. Выбывшее ОС остаётся в истории.                                                                        |
| 7.10  | Отчёты по основным средствам                  | Система формирует отчёты по наличию, движению и амортизации ОС. Отчёты доступны по организации, группе и периоду. Данные отчётов должны сходиться с бухгалтерским учётом. Отчёты используются для контроля и планирования инвестиций.                                                       |
