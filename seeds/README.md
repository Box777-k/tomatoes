# Seeds

Скрипты для заполнения базы данных начальными данными.

## Использование

Запуск всех seeds:

```bash
python -m seeds.seed
```

## Структура

- `seed.py` - основной скрипт с функциями для заполнения данных

## Принципы

1. Каждая функция seed проверяет наличие данных перед вставкой
2. Можно запускать многократно без дублирования
3. Используется транзакция с rollback при ошибках
4. Независимы от миграций Alembic

## Добавление новых seed

Создайте функцию в `seed.py`:

```python
def seed_something(db: Session):
    if db.query(Entity).count() > 0:
        print("Data already exists, skipping...")
        return
    
    # Ваш код вставки данных
    db.add_all(items)
    db.commit()
```

Добавьте вызов в `main()`.
